WESR Triquest — Provider Control Center

Provider dashboard

Create classes, select disciplines & roles to filter TR sections, choose taught sections, and publish your Course Code. Billing is per participant per class at first certificate issuance.

Create class

  1. Select disciplines (HV, LV, Mechanical, Hydraulic, Systems/Comms, Docs/Audit)
  2. Select roles (HV Operator, Switching Assistant, Work Planner, PCEI, PCWA, Delegated PCEI, Installer)
  3. Tick the TR sections taught (filtered by your choices)
  4. Review fixed question counts and locked difficulty mix
  5. Publish to receive a Course Code (e.g., WTRQ-HV-29F4)

Results

See per-section pass/fail; export CSV. Correct answers are not disclosed. Fail reports list missed item IDs for interview/clearance.

Billing (read-only)

  • Per participant per class at first endorsement issuance
  • Month-to-date usage & upcoming invoice
